# v0.8 事件牌文案与出图合并交接稿

## 1. 用途

本文件把每张事件牌的核心文案和单张 Gemini 出图提示词放在一起，方便按卡逐张生成。

- 一张牌对应一条完整 prompt
- 适配“Gemini 一次对话只生成一张图”的限制
- 出图完成后可直接进入 A4 打样或正式排版

## 2. 使用方法

1. 复制某一张牌对应的 `Gemini 单图 prompt`
2. 在 Gemini 中单独发起一次对话并粘贴
3. 生成后按建议文件名保存
4. 把图片与本页对应文案一起交给排版

## 3. 统一要求

- 不要生成任何文字、水印、英文字母标签
- 人物默认穿实验服
- 优先突出风险点，不追求复杂背景
- 构图统一为竖版卡牌友好的中近景
- 风格统一为“高校化学实验室安全桌游插画”

## 4. 文件命名规则

- 事件牌正面图：`assets/events/E##_front.png`
- 预览图：`assets/previews/E##_preview.png`
