# v0.8 Gemini 出图提示词包

## 1. 目的

本文件用于把 `v0.8` 的事件牌做成统一风格的卡牌插图和图标素材。

- 适用场景：Gemini 生成卡牌插图、封面图、图标草图
- 不适用场景：正式排版文案、规则说明页、可编辑矢量图标库

## 2. 使用原则

- 一次先确定统一风格，再批量出单卡
- 所有卡图都不要直接生成中文文字
- 优先做“信息清楚的实验室场景”，不要追求复杂背景
- 允许画出错误场景，但不要做夸张灾难和血腥效果
- 所有人物默认穿实验服；只有在事件需要表现防护缺失时才故意缺失某一项

## 3. 统一主提示词

把下面这一段作为所有卡牌插图的基础提示词：

```text
一套用于高校化学实验室安全桌游的统一风格卡牌插图，清晰、简洁、专业，偏平面插画与轻微写实结合，画面干净，构图稳定，强调实验室器材、风险点和操作环境，颜色分区清楚，适合桌游卡牌正面使用，不要文字，不要水印，不要夸张灾难场面，不要血腥，不要卡通搞笑风，背景控制在化学实验室内，突出单一事件焦点。
```

## 4. 统一负面约束

如果 Gemini 支持补充约束，可加：

```text
避免海报排版感，避免过多人物，避免复杂远景，避免错误的实验器材结构，避免多余装饰，避免模糊手部，避免把危险源画得太小，避免生成中文文字和英文字母标签。
```

## 5. 严重度视觉建议

- 低危：蓝灰、黄灰、浅冷色，氛围偏提醒
- 中危：橙色、暖灰，氛围偏紧张
- 高危：红色、深灰，氛围偏警戒


## 10. 实际出图建议

建议按以下顺序推进：

1. 先出 `E02 / E08 / E21`，确认器材表现是否靠谱
2. 再出 `E05 / E09 / E23`，确认风险点是否一眼能看懂
3. 最后出 `E13 / E15`，确认高危场景是否既清楚又不过度夸张
4. 风格稳定后，再批量补剩余 16 张事件牌
