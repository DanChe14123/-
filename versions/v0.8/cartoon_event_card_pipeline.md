# v0.8 卡通事件牌生成方案

## 1. 本轮目标

不依赖外部图片模型，直接在工作区生成统一风格的卡通事件牌正面 SVG。

## 2. 当前产出

- 生成脚本：`scripts/generate_event_card_svgs_v0_8.py`
- 输出目录：`assets/events/fronts/`
- 清单：`assets/previews/event_batch1_manifest.md`

## 3. 首批卡牌

本轮先做 8 张高代表性事件牌：

1. `E02`
2. `E05`
3. `E08`
4. `E09`
5. `E13`
6. `E15`
7. `E21`
8. `E23`

## 4. 风格要求

- 卡通风格
- 统一粗描边
- 扁平化色块加少量层次
- 竖版卡牌构图
- 卡面同时包含图像和正面文案

## 5. 当前不做

- PNG 导出
- 事件牌背面成图
- 任务牌和行动牌插图
- 最终印刷拼版

1. 剩余 16 张事件牌
2. 12 张任务牌
3. 12 类行动牌
