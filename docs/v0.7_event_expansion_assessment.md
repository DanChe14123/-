# v0.7 事件池扩容评估

## 1. 本轮目标

`v0.7` 的目标是把事件池从 `16` 张扩到 `24` 张，让 `v0.6` 已经展开的行动层获得更厚的对手盘。

本轮新增事件：

- `E17` 危化品领用记录与实物不一致
- `E18` 实验结束后设备未断电且记录缺失
- `E19` 装置软管连接松动并有渗漏
- `E20` 离心样品配平异常
- `E21` 旋蒸装置夹具松动或收集瓶偏斜
- `E22` 通风橱前窗开启过高导致气流失效
- `E23` 未知粉末散落并引发刺激性不适
- `E24` 钢瓶减压阀异常结霜并伴异响

正式数据目录：

- `C:\PKU\Term2\实验室安全作品大赛\data\v0.7\run_001_v0_7_event_expansion`

## 2. 关键结论

- 事件池扩容成功，没有破坏标准模式基线。
- `balanced+balanced` 无策略标准模式胜率为 `50.53%`，仍处于可试玩区间。
- 相比 `v0.6`，标准模式只小幅回落：`52.07% -> 50.53%`。
- 24 张事件全部稳定进入牌池，没有“挂名事件”。
- 新增 4 类行动合计选取率从 `34.10%` 提升到 `36.57%`。
- `secure` 选取率从 `2.18%` 提升到 `4.61%`，说明事件扩容确实给了窄用途行动更多空间。
- 当前主要残留问题不是标准模式，而是教学模式优势被压窄。

## 3. 基线表现

`balanced+balanced` 无策略：

- 标准模式：胜率 `50.53%`，平均进度 `3.400`
- 教学模式：胜率 `50.43%`，平均进度 `3.403`
- 挑战模式：胜率 `33.20%`，平均进度 `2.884`

对比 `v0.6` 标准模式无策略基线：

- 胜率：`52.07% -> 50.53%`
- 平均进度：`3.440 -> 3.400`
- 平均安全值：`3.453 -> 3.631`
- 平均升级标记：`0.537 -> 0.454`
- 平均隐患值：`1.280 -> 1.291`
- 平均换牌次数：`1.336 -> 1.195`

解释：

- 扩容后的压力更多体现在“推进变慢”，而不是“更容易崩盘”。
- 安全值和升级标记表现反而更稳，说明问题主要是节奏而不是生存。

## 7. 策略牌表现

标准模式下，`balanced+balanced` 的策略牌结果：

- `稳健管理`：胜率 `50.80%`
- `风险隔离`：胜率 `48.57%`
- `规范先行`：胜率 `47.63%`
- `效率优先`：胜率 `46.57%`

解释：

- 4 张策略牌都仍可用，但整体比 `v0.6` 更保守。
- 事件扩容后，不同策略仍主要体现在资源状态和失败结构上，体现在行动选取率上的差异仍然有限。

## 8. 当前判断

`v0.7` 已经把项目从“行动层展开的原型”推进到“事件池足够厚、能承接中体量玩法”的原型。

这版最重要的意义是：

- 24 张事件已经足够支撑更明显的局间变化
- 新行动不再担心缺少对应用武之地
- 后续可以停止扩系统，转入单卡精修和成品化准备

## 9. 建议的下一步

建议把下一版定义为 `v0.8`，目标是“精修与成品化准备版”：

1. 精修 `E18`、`T07` 等当前弱项
2. 适度回调教学模式，让它重新明显强于标准模式
3. 开始补事件牌背面的知识点文案和打印版原型
