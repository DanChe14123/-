# v0.6 行动层扩容评估

## 1. 本轮目标

`v0.6` 的目标不是继续扩大任务池，而是把行动层做宽，让不同任务和不同风险真正开始要求不同出牌方式。

本轮新增 4 类行动：

- `shutdown` 切断热源/电源
- `inspect` 检查并更换器材
- `secure` 固定与规范连接
- `evacuate` 撤离与警戒

同时做了两项配套调整：

- 公共手牌 `6 -> 8`
- 换牌逻辑改为更强调“事件专属关键解”

正式数据目录：

- `C:\PKU\Term2\实验室安全作品大赛\data\v0.6\run_001_v0_6_action_expansion`

## 2. 关键结论

- 行动层扩容成功，新增行动已经不是装饰，而是真正进入局内决策。
- `balanced+balanced` 无策略标准模式胜率为 `52.07%`，仍处在可试玩区间。
- 相比 `v0.5`，基线不降反升：`51.33% -> 52.07%`。
- `stop + report` 合计选取率从 `39.82%` 降到 `26.12%`，说明保守万能解已经被明显削弱。
- 新增 4 类行动合计选取率达到 `34.10%`，说明新行动不是摆设。
- 前 4 大行动的合计占比从 `67.63%` 降到 `51.12%`，说明出牌结构明显变宽。

## 3. 基线表现

`balanced+balanced` 无策略：

- 标准模式：胜率 `52.07%`，平均进度 `3.440`
- 教学模式：胜率 `54.40%`，平均进度 `3.542`
- 挑战模式：胜率 `31.63%`，平均进度 `2.775`

对比 `v0.5` 标准模式无策略基线：

- 胜率：`51.33% -> 52.07%`
- 平均进度：`3.408 -> 3.440`
- 平均安全值：`3.419 -> 3.453`
- 平均升级标记：`0.560 -> 0.537`
- 平均隐患值：`1.362 -> 1.280`

解释：

- `v0.6` 没有因为行动变复杂而变得更难。
- 反而在供牌和换牌逻辑调顺后，整体基线略有提升。

## 5. 任务池表现

标准模式、`balanced+balanced`、无策略下：

- 12 张任务全部稳定出现，出现次数约 `1409` 到 `1497`

较难完成的任务：

较易完成的任务：

## 6. 策略牌表现

标准模式下，`balanced+balanced` 的策略牌结果：

- `稳健管理`：胜率 `54.00%`
- `风险隔离`：胜率 `52.43%`
- `规范先行`：胜率 `52.23%`
- `效率优先`：胜率 `48.77%`

解释：

- 4 张策略牌仍然都可用，差距控制在合理区间。
- `效率优先` 依然是高上限高风险路线。
- 当前策略差异主要还体现在胜负结构和资源状态上，体现在行动选取率上的差异已经开始出现，但还不算特别大。

## 7. 当前判断

`v0.6` 已经把项目从“有完整任务池的原型”推进到“行动层真正展开的中等体量桌游原型”。

这一版最重要的意义不是数值略有提升，而是：

- 新行动已经被证明值得保留
- 老的万能行动已经被明显削弱
- 后续继续扩事件池时，有了足够宽的行动承接面

## 8. 建议的下一步

建议把下一版定义为 `v0.7`，目标是“事件池扩容版”：

1. 事件牌 `16 -> 24`
2. 优先补 `secure / inspect / evacuate` 相关事件
3. 让不同策略牌在事件应对上的差异更明显
