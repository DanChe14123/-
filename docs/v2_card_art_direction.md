# v2 卡面美术方向

## 1. 总原则

### Writing Guidance

v2 卡面目标是接近市面教育类桌游的完成度，而不是信息卡或 PPT 图标。美术应服务于 3 件事：

- 一眼看出实验室场景。
- 一眼看出风险源或正确动作。
- 中文全部后期排版，保证清晰。

## 2. 风格定位

推荐风格：

- 清爽卡通写实
- 学院感实验室
- 非 Q 版人物比例
- 柔和线条
- 明亮但不过度饱和
- 接近安全教育动画和轻策略桌游插画之间的质感

避免：

- 赛博朋克、暗黑惊悚、写实事故现场
- 过度夸张表情
- 复杂文字标签
- AI 生成的伪中文
- 太多器材堆叠导致主体不清

## 3. 图像层与文字层

硬规则：

- 插画层不出现任何中文。
- 编号、标题、风险等级、正文、图标全部后期排版。
- 插画中可出现空白标签、色块贴纸、符号图形，但不能出现可读文字。
- 卡面正文不压在插画上，只放在白底或浅灰信息块。

## 4. 卡面信息密度

正面只保留：

- 编号
- 卡名
- 类型或风险等级
- 主插画
- 1-2 个短信息块

背面保留：

- 正确处置
- 原因
- 知识点
- 复盘问题

详细解释进入说明书或复盘牌，不塞进卡面。

## 5. 五类卡视觉差异

| 卡类 | 主视觉 | 信息重点 |
| --- | --- | --- |
| 事件牌 | 风险正在发生 | 观察重点、错误倾向、正确处置 |
| 任务牌 | 正向实验操作 | 目标、优先行动、奖励 |
| 行动牌 | 单个动作示范 | 用途、适用边界 |
| 角色牌 | 人物立绘 | 能力、使用节奏 |
| 策略牌 | 团队路线场景 | 效果、代价、打法 |

## 7. 印刷与排版建议

### Writing Guidance

- 建议竖版卡牌，圆角外框，顶部色带。
- 标题使用黑色粗体，放纯白区域。
- 正文用 2-4 行短句，字号不小于打印可读下限。
- 插画四角留安全区，避免裁切主体。
- 所有源文件应保留插画层、文字层和结构化数据。
