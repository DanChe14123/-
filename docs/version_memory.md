# 版本速记

## v0.1

- 定位：最小原型
- 结果：标准模式过难，`balanced+balanced` 胜率约 `6%`

## v0.2

- 定位：可试玩 Demo
- 结果：标准模式进入可玩区间，`balanced+balanced` 胜率 `56.27%`

## v0.3

- 定位：完整度增强版
- 特征：任务绑定、隐患值、任务奖励
- 结果：像桌游了，`balanced+balanced` 标准模式胜率 `53.83%`

## v0.4

- 定位：流派起点版
- 特征：首发 4 张实验室策略牌
- 结果：
  - 无策略基线 `53.83%`
  - 四张策略牌胜率区间约 `49.20%` 到 `54.97%`
  - 已有流派起点，但行动差异还不够大

## v0.5

- 定位：任务扩容版
- 特征：
  - 任务牌 `8 -> 12`
  - 新增 `task_stats.csv`
  - 保持 `v0.4` 机制框架不变，专门验证内容体量
- 结果：
  - `balanced+balanced` 无策略标准模式胜率 `51.33%`
  - 教学模式胜率 `55.87%`
  - 挑战模式胜率 `26.13%`
  - 12 张任务全部稳定进入牌池
  - 任务完成率区间约 `50.34%` 到 `70.82%`

## v0.6

- 定位：行动层扩容版
- 特征：
  - 行动牌 `8 -> 12`
  - 新增 `shutdown / inspect / secure / evacuate`
  - 公共手牌 `6 -> 8`
  - 换牌逻辑改为更强调“事件专属关键解”
- 结果：
  - `balanced+balanced` 无策略标准模式胜率 `52.07%`
  - 教学模式胜率 `54.40%`
  - 挑战模式胜率 `31.63%`
  - 新增 4 类行动合计选取率 `34.10%`
  - `stop + report` 合计选取率从 `39.82%` 降到 `26.12%`
  - 行动结构明显变宽，前 4 大行动占比从 `67.63%` 降到 `51.12%`

## v0.7

- 定位：事件池扩容版
- 特征：
  - 事件牌 `16 -> 24`
  - 新增 `E17-E24`
  - 重点补 `secure / inspect / evacuate`
  - 任务与事件映射同步扩容
- 结果：
  - `balanced+balanced` 无策略标准模式胜率 `50.53%`
  - 教学模式胜率 `50.43%`
  - 挑战模式胜率 `33.20%`
  - 24 张事件全部稳定进入牌池
  - 新增 4 类行动合计选取率 `36.57%`
  - `secure` 选取率从 `2.18%` 提升到 `4.61%`
  - `stop + report` 合计选取率进一步降到 `25.49%`

## v0.8

## 当前总体判断

- `v0.1` 解决了“有没有核心循环”
- `v0.2` 解决了“能不能稳定试玩”
- `v0.3` 解决了“像不像一个桌游”
- `v0.4` 解决了“有没有流派起点”
- `v0.5` 解决了“内容体量是否足够支撑更完整的任务池”
- `v0.6` 解决了“行动层是否足够宽，能不能支撑真正的玩法分化”
- `v0.7` 解决了“事件池是否足够厚，能不能承接已经展开的行动层”
- `v0.8` 解决了“能不能在不改系统的前提下，把弱项和教学模式调到可用状态”

## 下一阶段重点

- 保持 `v0.8` 作为稳定玩法基线
- 按视觉圣经继续把全部卡面落成正式版
