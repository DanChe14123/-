# NotebookLM 视频资料包说明与使用方法

## 1. 这套资料包是做什么的

这套资料包的目标不是单纯介绍一个桌游，而是帮助 NotebookLM 学会以下四件事：

1. 这件参赛作品到底是什么  
2. 这套桌游为什么适合化学实验室安全培训  
3. 这套桌游的核心玩法和教学逻辑是什么  
4. 这套作品是如何经过多轮测试和迭代才稳定下来的

最终希望 NotebookLM 输出的不是空泛宣传，而是一支面向化学院老师和评委的、结构清晰、逻辑扎实、具有教育意义的 5 分钟宣传视频脚本。

## 2. 建议上传到 NotebookLM 的顺序

建议按下面的顺序上传，先让 NotebookLM 建立“作品是什么”的概念，再让它理解玩法、视觉和数据。

### 第一组：先上传本资料包

1. `00_资料包说明与使用方法.md`

3. `02_桌游规则完整说明_v0.8.md`
4. `03_桌游视觉版面与卡面说明.md`
5. `04_桌游数据与版本演化说明.md`

   - `E02.png`

上面两份 PDF 的作用不是替代本资料包，而是作为更正式的补充材料，让 NotebookLM 在生成脚本时能引用更加完整的项目表达。

## 3. 使用 NotebookLM 的推荐流程

不要一上来就让 NotebookLM 直接输出最终视频全文稿。更稳妥的做法是分四步走。

### 第一步：让它先总结作品

先让 NotebookLM 回答：

- 这件作品是什么
- 核心玩法是什么
- 核心教育价值是什么
- 面向谁使用

如果这一步说不清楚，后面的脚本基本一定会跑偏。

### 第二步：让它先出 5 分钟视频大纲

要求 NotebookLM 输出：

1. 5 分钟视频的分段结构
2. 每一段各讲什么
3. 每一段应该配什么画面
4. 哪些地方展示卡牌，哪些地方展示实验室场景，哪些地方展示数据

这一步的目标不是文案优美，而是逻辑正确。

### 第三步：让它输出完整解说词

要求 NotebookLM 在已经确定结构的基础上，输出：

1. 完整旁白稿
2. 每段时长建议
3. 每段画面建议
4. 适合出现在画面上的简短标题

### 第四步：让它输出制作版脚本

在第三步基础上，再让它整理成：

1. 分镜头脚本
2. 旁白文本
3. 画面素材调用建议
4. 哪些地方要放数据图
5. 哪些地方要放卡牌样板图

## 4. 建议你直接对 NotebookLM 说的话

你可以在上传完资料后，先直接输入下面这段话：

> 请先不要直接生成最终视频文案。请先根据我上传的资料，总结这件作品是什么、核心玩法是什么、它为什么适合化学实验室安全培训、为什么它对化学院老师是有价值的。然后请给我一个面向高校化学院评委的 5 分钟宣传视频结构大纲。

等它给出大纲后，再输入：

> 现在请根据刚才的大纲，生成一版完整的 5 分钟宣传视频脚本。要求语言正式、清楚、适合化学院老师理解，不要使用太多桌游黑话，不要写成商业游戏广告。

如果它输出太空泛，再输入：

> 请进一步加强以下内容：1. 桌游的教学逻辑；2. 实验室真实性；3. 多版本迭代和数据验证；4. 作品适合在新生培训中落地使用。

## 5. 使用这套资料包时的注意事项

1. 不要只上传卡图，不上传规则和数据。  
只上传卡图，NotebookLM 容易把作品理解成“视觉化安全宣传牌组”，而不是“有完整玩法的协作培训桌游”。

2. 不要只上传规则，不上传视觉资料。  
只上传规则，NotebookLM 容易写成纯规则说明，而不是适合做宣传视频的脚本。

3. 不要只上传数据，不上传项目定位。  
只上传数据，NotebookLM 会知道你做了很多迭代，但不一定知道该如何把这些数据讲成“可信、实用、适合教学”的故事。

4. 最重要的是让它先理解“这是什么”，再让它写“怎么宣传”。  

## 6. 这套资料包最终应该帮助 NotebookLM 输出什么

一个理想的最终输出，应该至少包括：

1. 5 分钟视频总结构
2. 完整旁白稿
3. 每段画面建议
4. 卡牌图片和数据图的插入位置建议
5. 面向化学院老师的表达方式
6. 强调教育意义、实用性和项目迭代可信度的结尾

如果 NotebookLM 输出的内容缺少这些部分，就说明它还没有真正学会这套作品。
