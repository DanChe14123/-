# v0.8 精修评估

## 1. 本轮目标

`v0.8` 的目标不是继续扩卡池，而是在 `v0.7` 已经成型的框架上，修正最明显的弱项并把教学模式调回真正的入门版本。

本轮重点：

- 修正 `T07` 与 `E23` 的任务映射错位
- 改善低危 `shutdown / secure` 事件的换牌获取
- 回调教学模式参数与节奏

当前有效正式数据：

- `C:\PKU\Term2\实验室安全作品大赛\data\v0.8\run_002_v0_8_refinement_teaching3`

## 2. 关键结论

- 精修成功，标准模式不但没有变差，反而从 `50.53%` 提升到 `53.10%`。
- 教学模式被重新拉开，`balanced+balanced` 胜率达到 `81.10%`。
- `T07` 完成率从 `v0.7` 的 `47.14%` 回升到 `58.80%`。
- 行动结构基本不变，说明本轮修正没有破坏 `v0.6/v0.7` 建立起来的玩法分化。
- `E18` 仍然是后续值得继续精修的对象，但已不再是系统级阻碍。

## 3. 基线表现

`balanced+balanced` 无策略：

- 标准模式：胜率 `53.10%`，平均进度 `3.443`
- 教学模式：胜率 `81.10%`，平均进度 `3.500`
- 挑战模式：胜率 `33.80%`，平均进度 `2.893`

对比 `v0.7` 标准模式无策略基线：

- 胜率：`50.53% -> 53.10%`
- 平均进度：`3.400 -> 3.443`
- 平均安全值：`3.631 -> 3.682`
- 平均升级标记：`0.454 -> 0.453`
- 平均隐患值：`1.291 -> 1.293`

解释：

- 本轮主要解决的是推进效率，而不是生存问题。
- 标准模式的整体体感更稳了。

## 4. 教学模式表现

`balanced+balanced` 教学模式：

- 胜率 `81.10%`
- 平均安全值 `5.404`
- 平均隐患值 `1.043`
- 失败主因仍以 `progress_shortfall` 为主，但数量大幅下降

解释：

- `v0.8` 的教学模式已经不再和标准模式混在一起。
- 这是一个明显更友好的训练模式，适合新手学习与体验。
